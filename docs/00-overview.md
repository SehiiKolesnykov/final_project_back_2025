# Beck Spring API Documentation

## Overview

Beck Spring API - це RESTful веб-сервіс для соціальної мережі з функціоналом постів, коментарів, лайків, підписок та чату в реальному часі.

## Base URL

- **Production**: `https://step-project-api.onrender.com`

## Authentication

Більшість ендпоінтів вимагають JWT автентифікації.

### Отримання токену

Використовуйте наступні ендпоінти для отримання токену:
- `POST /api/auth/login` - для входу
- `POST /api/auth/register` - для реєстрації

### Використання токену

Надсилайте токен одним з двох способів:

1. **HTTP Header**:
   ```
   Authorization: Bearer <token>
   ```

2. **HttpOnly Cookie**:
   ```
   Cookie: jwt=<token>
   ```

## CORS

API дозволяє запити з будь-якого джерела (`*`).

## Error Handling

Помилки повертаються з відповідними HTTP статус кодами та JSON тілом:

```json
{
  "error": "Повідомлення про помилку"
}
```

### Типові коди помилок

- `400 Bad Request` - невалідні дані в запиті
- `401 Unauthorized` - відсутня або невалідна автентифікація
- `403 Forbidden` - недостатньо прав для виконання операції
- `404 Not Found` - ресурс не знайдено
- `409 Conflict` - конфлікт даних (наприклад, користувач вже існує)

## Pagination

Для ендпоінтів, що повертають списки, використовується пагінація:

```
?page=0&size=20
```

### Параметри

- `page` - номер сторінки (починається з 0)
- `size` - кількість елементів на сторінці
- **За замовчуванням**: `page=0`, `size=20`

### Формат відповіді

Пагіновані відповіді включають:
- `content` - масив елементів
- `page` - поточна сторінка
- `size` - розмір сторінки
- `totalElements` - загальна кількість елементів
- `totalPages` - загальна кількість сторінок
- `hasNext` - чи є наступна сторінка

## Data Format

Всі запити та відповіді використовують **JSON** формат.

### Дати та час

- **Формат дати народження**: `dd.MM.yyyy` (наприклад, `01.01.1990`)
- **Формат дати оновлення**: `yyyy-MM-dd` (наприклад, `1990-01-01`)
- **Формат дати/часу в відповідях**: ISO 8601 (наприклад, `2026-01-10T14:48:00`)

## Required Fields

У документації обов'язкові поля позначені зірочкою (`*`).
